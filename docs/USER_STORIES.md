# User Stories – Student Assignment Tracker

---

## User Stories Table

| Story ID | User Story                                                                              | Acceptance Criteria                                          | Priority |
| -------- | --------------------------------------------------------------------------------------- | ------------------------------------------------------------ | -------- |
| US-001   | As a student, I want to register an account so that I can access the system             | Valid email required; account created successfully           | High     |
| US-002   | As a user, I want to log in so that I can access my dashboard                           | Valid credentials grant access; invalid credentials rejected | High     |
| US-003   | As a lecturer, I want to create assignments so that students can access coursework      | Assignment saved with title, description, and due date       | High     |
| US-004   | As a student, I want to view assignments so that I can see my tasks                     | Assignments displayed with all details                       | High     |
| US-005   | As a lecturer, I want to update assignments so that I can correct or modify information | Changes saved successfully                                   | Medium   |
| US-006   | As a lecturer, I want to delete assignments so that outdated tasks are removed          | Assignment removed from system                               | Medium   |
| US-007   | As a student, I want to track deadlines so that I can manage my time effectively        | Deadlines clearly displayed; overdue flagged                 | High     |
| US-008   | As a student, I want to mark assignments as completed so that I can track progress      | Status updates successfully                                  | Medium   |
| US-009   | As a student, I want to receive notifications so that I don’t miss deadlines            | Notifications sent 24 hours before due date                  | High     |
| US-010   | As an admin, I want to manage users so that system access is controlled                 | Users can be added/removed successfully                      | Medium   |
| US-011   | As a system admin, I want user data encrypted so that security is maintained            | Data stored using secure encryption                          | High     |
| US-012   | As a student, I want fast system responses so that I can efficiently use the platform   | System responds within 2 seconds                             | High     |
